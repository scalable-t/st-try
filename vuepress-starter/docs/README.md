# Hello VuePress

[[toc]]

随便尝试一下 `VuePress` 文档管理系统。

## 验证

需要验证一些目标：

- Y 离线使用是否方便
- Y 是否有搜索功能
- Y 搜索功能是否可以离线使用 (只能标题和 tag 搜索)
- Y 目录组织能力是否好用
- Y 是否可以引用（复用）
- N 能否导出 pdf
- N 能否导出 docx
- Y 是否好看
- Y 是否能改主题 

## 结论

`VuePress` 用作技术文档非常非常好用，界面美观，移动端自适应，定制性很强，目录结构方便，基础功能完善，可静态化导出，很现代化。

只是不支持导出传统格式而已。
